# Edge-AI-Powered-Real-Time-Sepsis-Prediction-with-Secure-Multi-Sensor-Fusion-for-ICU-Care
Edge-AI Powered Real-Time Sepsis Prediction with Secure Multi-Sensor Fusion for ICU Care uses Edge AI to analyze fused vital signs at the bedside and predict sepsis risk in real time. The system delivers low-latency alerts with secure data handling to support faster and safer ICU decision-making.
